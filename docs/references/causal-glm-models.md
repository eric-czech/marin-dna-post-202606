### CLM models

Decoder only, autoregressive, causal genomic models:

| Model / family | Architecture / objective evidence | Scope | Sources |
|---|---|---|---|
| **GenSLM / GenSLMs — GPT checkpoints only** | GenSLM used GPT-style implementations; Cerebras says the trained GenSLM models used GPT-2 and GPT-J implementations with a genomic dataloader/config changes. | Prokaryotic genes + SARS-CoV-2 | [Cerebras GenSLM post](https://www.cerebras.ai/blog/genomics-in-unparalleled-resolution-cerebras-wafer-scale-cluster-trains-large-language-models-on-the-full-covid-genome-sequence), [GenSLM paper](https://pmc.ncbi.nlm.nih.gov/articles/PMC9709791/) |
| **DNAGPT** | Described as enhancing a classic GPT model and trained on mammalian DNA with causal/generative DNA modeling plus auxiliary tasks. | Mammalian DNA | [DNAGPT arXiv](https://arxiv.org/abs/2307.05628) |
| **METAGENE-1** | A 7B-parameter autoregressive Transformer language model trained on metagenomic DNA/RNA; uses causal language modeling with a LLaMA-style setup. | Metagenomic DNA/RNA from wastewater | [METAGENE-1 arXiv HTML](https://arxiv.org/html/2501.02045v1) |
| **GenomeOcean** | Hugging Face card says GenomeOcean is trained with Causal Language Modeling and loaded with `AutoModelForCausalLM`; model metadata tags it as Mistral-style. | Microbial / metagenomic assemblies | [GenomeOcean-100M Hugging Face](https://huggingface.co/DOEJGI/GenomeOcean-100M), [GenomeOcean OpenReview](https://openreview.net/forum?id=c8sEgxG2c0) |
| **GENERator / GENERATOR-v2** | GENERATOR is a Transformer-decoder DNA LM pretrained with an autoregressive next-token prediction objective; GENERATOR-v2 keeps the Transformer decoder architecture with 6-mer tokenization and 98k context. | Eukaryotic and prokaryotic DNA | [GENERATOR arXiv HTML](https://arxiv.org/html/2502.07272v4), [GENERATOR GitHub](https://github.com/GenerTeam/GENERator), [GENERATOR-v2 bioRxiv](https://www.biorxiv.org/content/10.64898/2026.01.27.702015v2.full), [GENERATOR-v2 HF model](https://huggingface.co/GenerTeam/GENERator-v2-eukaryote-1.2b-base) |
| **Carbon** | Carbon is a family of causal language models; Carbon-3B is a decoder-only autoregressive genomic foundation model and the model card says it is a standard Hugging Face causal LM / stock `LlamaForCausalLM`. | DNA/RNA, eukaryote-heavy | [Carbon GitHub](https://github.com/huggingface/carbon), [Carbon-3B Hugging Face](https://huggingface.co/HuggingFaceBio/Carbon-3B), [Carbon bioRxiv](https://www.biorxiv.org/content/10.64898/2026.05.22.727119v1.full-text) |
| **Gene42** | Gene42 uses a decoder-only LLaMA-style autoregressive Transformer with dense self-attention and long-context pretraining. | Human + some multispecies genomic DNA | [Gene42 arXiv](https://arxiv.org/abs/2503.16565), [Gene42 arXiv HTML](https://arxiv.org/html/2503.16565v1) |
